🏥 Hospital Patient Registry

📌 Overview

Hospital Patient Registry is a simple web-based application developed using Node.js, HTML, and CSS. The system allows users to register patient details, store records in a text file, and view patient information through a browser interface.

🚀 Features

- Register new patients
- Store patient records in "patient_registry.txt"
- View all registered patients
- Simple and user-friendly interface
- File handling using Node.js "fs" module

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- File System (fs) Module

📂 Project Structure

hospital-patient-registry/

├── index.html

├── style.css

├── server.js

├── patient_registry.txt

└── README.md

📋 Patient Information Stored

- Patient Name
- Registration Date
- Disease / Medical Condition

▶️ How to Run the Project

Step 1: Install Node.js

Download and install Node.js from the official website.

Step 2: Open Project Folder

Open the project folder in VS Code.

Step 3: Run the Server

node server.js

Step 4: Open Browser

http://localhost:3000

Step 5: Register Patient

Enter patient details and click Register Patient.

Step 6: View Records

Click View Patients to see all saved patient records.
